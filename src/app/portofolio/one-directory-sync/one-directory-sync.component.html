One Directory Sync

Characteristics

Project Type: Enterprise

Client: OpenLM, Israel

Period: September 2021 - July 2022

Overview

       The main objective of the OneDirectorySync project is to provide a comprehensive solution that enables seamless synchronization of changes from an organization's primary directory to all connected cloud services. By doing so, the project aims to streamline IT efforts in data protection and security management, avoiding potential security risks and ensuring data integrity.

Key Features

  • Delete Users (No Notification): This option enables the deletion of users from the primary directory without sending any notifications. It ensures swift removal of user accounts without generating any user-facing notifications.
  • Delete Users and Notify: With this feature, user deletion from the primary directory triggers a notification to inform affected users. Notifications can be sent via email, ticketing system integration, or push notifications, ensuring that users are aware of the account removal.
  • Notify User for Deletion: In this mode, instead of deleting the user, OneDirectorySync generates a notification to inform the user that they should be deleted. It prompts the IT department to take action and manually remove the user account.
  • Notification Methods: OneDirectorySync supports multiple notification methods, including email, integration with IT ticketing systems, and push notifications. These methods ensure effective communication with IT Team and facilitate seamless user management.

Achievements

  • The project improved data synchronization efficiency by reducing the time required for manual updates. This resulted in a 40% reduction in manual synchronization efforts, saving approximately 200 hours per month.
  • The project ensured data consistency across cloud services, reducing data discrepancies by 95%. This resulted in fewer errors and improved data accuracy, enhancing decision-making processes within the organization.
  • The project strengthened data security by automating the removal of user accounts from connected cloud services. This reduced the risk of unauthorized access and potential data breaches, resulting in an estimated cost savings of $50,000 in potential security incidents.
