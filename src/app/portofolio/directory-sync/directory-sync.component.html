Directory Sync

Characteristics

Project Type: Enterprise

Client: OpenLM, Israel

Overview

       The Directory Sync enables organizations to synchronize their directory service with OpenLM database to ensure consistent and synchronized user information and access control. Directory synchronization is performed between directories that adhere to the LDAP protocol, such as Active Directory, ApacheDS, or eDirectory.
